## Cara deploy ke Vercel

1. Upload semua isi folder ke repository GitHub baru.
2. Buka Vercel dan pilih **Add New Project**.
3. Import repository tersebut.
4. Framework preset dapat menggunakan **Other**.
5. Deploy tanpa environment variable.

Website ini tidak memerlukan npm, database, atau API.
