PROFILE

Built from real operations, not assumptions.

My work sits between warehouse operations and software delivery. I translate daily constraints—stock accuracy, FIFO, batch history, approval delays, manual reports—into systems that people can actually use on the floor.

I contribute across process mapping, data structure, UI/UX, automation logic, testing, and deployment. The result is not only a dashboard, but an operational workflow with clear controls and traceable data.

01

Operational first

The system follows how work is performed, including exceptions and controls.

02

Traceable by design

03

Complex logic stays behind a fast, mobile-friendly interface.
